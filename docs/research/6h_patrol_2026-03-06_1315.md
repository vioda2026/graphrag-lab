# GraphRAG 研究巡检报告（6 小时）— 2026-03-06 13:15 (Asia/Shanghai)

## A) 用户简洁摘要（5-10 条要点）

1. **GraphRAG-Bench 基准测试** (ICLR'26 接收)
   - 官方评测基准：GraphRAG-Bench (Novel) + GraphRAG-Bench (Medical) 双榜单
   - 四难度层级：Fact Retrieval → Complex Reasoning → Contextual Summarization → Creative Generation
   - 评测维度：Accuracy, ROUGE-L, Coverage, Factual Score

2. **LinearRAG** (ICLR'26 接收)
   - 无关系图构建方法，追求高效 GraphRAG
   - 适合大规模语料场景

3. **LogicRAG** (AAAI'26 接收)
   - 逻辑推理增强的 GraphRAG 方法

4. **核心研究方向汇总**：
   - 图结构创新：core-based 社区分解、attributed community hierarchy
   - 图探索策略：query-centric retrieval, multi-hop reasoning, GNN-enhanced retrieval
   - 检索增强：hybrid retriever, multi-round retriever, post-retrieval refinement

5. **代表性新论文** (2025-2026)：
   - ArchRAG: Attributed Community-based Hierarchical RAG
   - KET-RAG: Cost-Efficient Multi-Granular Indexing
   - PIKE-RAG: Specialized Knowledge and Rationale Augmented
   - E²GraphRAG: Streamlining Graph-based RAG for Efficiency
   - SUBQRAG: Sub-Question Driven Dynamic Graph RAG

6. **医疗领域 GraphRAG**：
   - Medical Graph RAG (CVPR'25): 安全医疗 LLM
   - MedRAG: Knowledge Graph-Elicited Reasoning for Healthcare

7. **开源项目动态**：
   - Microsoft GraphRAG: 官方实现持续更新
   - LightRAG: 简单快速实现
   - DIGIMON: 统一模块化框架
   - Fast GraphRAG: 自适应用例

8. **关键发现**：
   - GraphRAG 并非总是优于传统 RAG
   - 图结构在层级知识检索和深度上下文推理场景表现更佳
   - 需要针对具体任务选择合适的 GraphRAG 变体

9. **评测趋势**：
   - 从单一 Accuracy 向多维度评估转变
   - 强调事实一致性 (Factual Score) 和覆盖率 (Coverage)
   - 开始关注幻觉检测 (Hallucination Detection)

10. **下一步研究方向**：
    - 自适应图构建 (Adaptive Graph Construction)
    - 多模态 GraphRAG (Multi-modal GraphRAG)
    - 持续学习与非参数记忆 (Non-Parametric Continual Learning)

## B) 详尽结构化记录

### 1. Benchmark 进展

#### GraphRAG-Bench (ICLR'26)
- **论文**: "When to use Graphs in RAG: A Comprehensive Analysis for Graph Retrieval-Augmented Generation"
- **作者**: Xiang Zhishang, Wu Chuanjie, Zhang Qinggang, et al. (Xiamen University, PolyU)
- **评测数据集**:
  - GraphRAG-Bench (Novel): 文学/虚构内容
  - GraphRAG-Bench (Medical): 医疗/健康内容
- **任务难度分级**:
  | Level | 任务类型 | 示例 |
  |-------|----------|------|
  | 1 | Fact Retrieval | "Mont St. Michel 位于法国哪个地区？" |
  | 2 | Complex Reasoning | "Hinze 与 Felicia 的协议如何影响英格兰统治者的看法？" |
  | 3 | Contextual Summarization | "John Curgenven 作为康沃尔船夫的角色是什么？" |
  | 4 | Creative Generation | "将亚瑟王与 John Curgenven 的比较改写成新闻报道" |
- **评测指标**: Accuracy, ROUGE-L, Coverage, Factual Score
- **支持框架**: LightRAG, Microsoft GraphRAG, 等
- **GitHub**: https://github.com/GraphRAG-Bench/GraphRAG-Benchmark

### 2. 图结构创新

#### 2.1 Core-based Hierarchies (2026-03-05)
- **论文**: Core-based Hierarchies for Efficient GraphRAG
- **作者**: Jakir Hossain, Ahmet Erdem Sarıyüce
- **创新点**: 基于 core decomposition 的社区分解优化层级索引
- **优势**: 减少索引构建成本，提升检索效率

#### 2.2 Attributed Community Hierarchy (ArchRAG)
- **论文**: ArchRAG: Attributed Community-based Hierarchical Retrieval-Augmented Generation (arXiv 2025)
- **创新点**: 带属性的社区层级结构
- **优势**: 更细粒度的知识组织

#### 2.3 Multi-Granular Indexing (KET-RAG)
- **论文**: KET-RAG: A Cost-Efficient Multi-Granular Indexing Framework for Graph-RAG (arXiv 2025)
- **创新点**: 多粒度索引框架
- **优势**: 成本效益优化

### 3. 图探索策略创新

#### 3.1 Query-Centric Retrieval
- **论文**: Query-Centric Graph Retrieval Augmented Generation (arXiv 2025)
- **创新点**: 以查询为中心的图检索策略
- **优势**: 提升检索相关性

#### 3.2 Multi-Agent GraphRAG
- **论文**: Multi-Agent GraphRAG: A Text-to-Cypher Framework for Labeled Property Graphs (arXiv 2025)
- **创新点**: 多智能体协作的 Text-to-Cypher 框架
- **优势**: 支持标签属性图的复杂查询

#### 3.3 SUBQRAG
- **论文**: SUBQRAG: SUB-QUESTION DRIVEN DYNAMIC GRAPH RAG (arXiv 2025)
- **创新点**: 子问题驱动的动态图 RAG
- **优势**: 分解复杂查询，逐层推理

### 4. 检索增强技术

#### 4.1 Hybrid Retriever
- **论文**: HybGRAG: Hybrid Retrieval-Augmented Generation on Textual and Relational Knowledge Bases (arXiv 2024)
- **创新点**: 文本 + 关系知识库混合检索
- **优势**: 结合两种知识源的优势

#### 4.2 GNN-Enhanced Retrieval
- **论文**: Graph Neural Network Enhanced Retrieval for Question Answering of LLMs (arXiv 2024)
- **创新点**: GNN 增强的检索机制
- **优势**: 利用图结构信息进行更精准的检索

#### 4.3 Multi-Round Retriever
- **论文**: Multiple works on iterative retrieval refinement
- **创新点**: 多轮检索 refinement
- **优势**: 逐步逼近最优答案

### 8. 重要发现与趋势

1. **GraphRAG 有效性条件**：
   - 在层级知识检索场景优于传统 RAG
   - 在深度上下文推理任务表现更佳
   - 在简单事实检索任务可能不如传统 RAG

2. **评估趋势**：
   - 从单一 Accuracy 向多维度评估转变
   - 强调事实一致性 (Factual Score)
   - 关注幻觉检测 (Hallucination Detection)

3. **技术趋势**：
   - 自适应图构建 (Adaptive Graph Construction)
   - 多模态 GraphRAG (Multi-modal GraphRAG)
   - 持续学习 (Non-Parametric Continual Learning, HippoRAG2)
